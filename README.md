# EdibleFlowers
食用花卉

## 食用花卉
- 对目前已有的花卉菜肴、花卉饮品，进行汇总、分类，展示其营养价值。

## 食用花卉识别
- 对各类食用花卉能够进行准确识别，分析该类食品得营养成分。

## 食用花卉寓意
- 对食用花卉的文化进行宣传，具体形式未定，将食用花卉文化发扬光大。
